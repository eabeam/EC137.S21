Assignment

  1. Install R

    You can download R from CRAN. Select your operating system.

    Windows users: Select base, and then download and install R

    Mac users: Download and install R-4.04.pkg

    If you get an error message, then you should hold down the control key while clicking the package, then select "open" (still holding the key), and once you see a warning message, select "open."

  2. Install RStudio

  3. Complete Programming Basics tutorial (Note: you will need to create a free RStudio Cloud account). In terms of time budgeting, note that this will involve you watching 7 videos that are 1-2 minutes long. There will also be short activities you can do, in which you type into boxes and click "run code" to test it. You should do these too!

  4. Open RStudio on your computer.

  5. Type the following code in your Console window and press Enter: sqrt(pi). Look, you calculated !

  6. Install the tidyverse! Type the following code in your Console window and press Enter: 

    xxxxxxxxxx
    install.packages("tidyverse")

    It incldues fun friends like dplyr (pronounced "dee-plier") and more!

  7. Tell R to use dplyr. Type the following code in your Console window and press Enter: 

    xxxxxxxxxx
    library(dplyr)
